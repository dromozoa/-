# 効果音

[効果音ラボ](https://soundeffect-lab.info/sound/button/)からダウンロードした素材のラウドネスを調整し、サウンドスプライトを作成する。

| 名称              | ファイル名 |
|-------------------|------------|
| 決定ボタンを押す2 | select     |
| キャンセル1       | cancel     |
| ビープ音1         | beep       |
| 警告音1           | alert      |
| 成功音            | trophy     |
